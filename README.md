# nidhishhhh.github.io

🌐 Personal landing page for **drp.ai** — an AI-powered fashion visual search engine.

## About

This repository hosts the landing page for drp.ai, a fashion discovery platform that lets users:
- Upload outfit photos
- Find visually similar products from top brands
- Get styled recommendations with real prices
- Build complete looks in one click

## Tech Stack

- HTML5 / CSS3
- GitHub Pages (hosting)
- drp.ai API (backend)
